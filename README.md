# 🇨🇳 中华人民共和国超级计算器 v1.0

**代号: 红龙** 🐉

---

## 🧮 简介

一款完全中文化界面的超级计算器。  
使用中文数字输入，计算结果显示也是中文数字。

**示例:**

```
   一二三
+ 四五六
= 五七九
```

---

## 🚀 快速启动

### 方式一 — 直接运行

```bash
cd dist/
./习近平计算器.exe
```

### 方式二 — 源码运行（需安装 Go）

```bash
go build -o 习近平计算器.exe .
./习近平计算器.exe
```

## 🎮 使用说明

### 按钮

| 按钮 | 功能 |
|--------|--------|
| **七** **八** **九** … **零** | 输入中文数字 |
| **＋** | 加法 |
| **−** | 减法 |
| **×** | 乘法 |
| **÷** | 除法 |
| **＝** | 计算结果 |
| **清除** | 全部清除 |
| **←** | 删除末位 |
| **．** | 小数点 |

### 键盘快捷键

| 按键 | 功能 |
|---------|----------|
| `+` `-` `*` `/` | 运算符 |
| `一二三四五六七八九` | 直接输入中文数字 |
| `Enter` | 计算结果 |
| `Backspace` | 删除末位 |
| `Escape` | 全部清除 |

### 主席模式

点击 **主席** 按钮 — 界面变为全红，
毛泽东画像出现，并播放 **《东方红》**。

## 🔧 技术细节

- **语言:** Go
- **GUI:** walk (Windows 原生控件)
- **构建:** go build
- **图片:** walk 原生支持
- **音频:** ffplay (外部进程播放)
- **所有变量名均为中文**

### 系统要求

- **Windows** (仅支持 Windows)
- **Go 1.20+** (编译需要)
- **ffplay** (可选，用于主席模式音乐)
